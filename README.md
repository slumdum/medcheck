# 🩺 MedCheck — AI Symptom Checker (Gemini Free Version)

## ▶️ How to run (3 steps only)

### Step 1 — Make sure Node.js is installed
Open Command Prompt and type:
```
node --version
```
If you see a version number, you're good. If not, download from https://nodejs.org

### Step 2 — Start the server
1. Extract this zip to a folder (e.g. C:\medcheck-gemini)
2. Open Command Prompt inside that folder
   (Click the address bar in File Explorer, type `cmd`, press Enter)
3. Run this command:
```
node server.js
```
You should see:
```
✅ MedCheck server running with Gemini (Free)!
👉 Open your browser and go to: http://localhost:3000
```

### Step 3 — Open the app
Go to your browser and visit: http://localhost:3000

🎉 Done! Your symptom checker is running for FREE using Gemini.

---

## ⚠️ Important
- Keep the Command Prompt window open while using the app
- Your API key is already inside server.js — no changes needed
